Prognosen / Forecasts

Die App sammelt anonymisierte Auslastungsdaten aller Nutzer und speichert diese auf einem externen Server (Supabase), um daraus gemeinschaftliche Durchschnittswerte und Prognosen zu berechnen. Dabei wird ein zufällig generierter, anonymer Geräte-Hash zur Deduplizierung verwendet – es werden keine personenbezogenen Daten übermittelt. Diese Prognosen basieren auf historischen Daten und stellen keine Garantie für die tatsächliche zukünftige Auslastung dar.
The app collects anonymized occupancy data from all users and stores it on an external server (Supabase) to calculate community-based averages and forecasts. A randomly generated, anonymous device hash is used for deduplication – no personal data is transmitted. These forecasts are based on historical data and do not guarantee actual future occupancy levels.
